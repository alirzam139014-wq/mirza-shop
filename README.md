# 🏪 Mirza Shop (میرزا شاپ)

نرم‌افزار مدیریت محصوالت فروشگاه - کاملاً آفلاین

## 📋 درباره پروژه

Mirza Shop یک نرم‌افزار حرفه‌ای برای ثبت، مدیریت، جستجو و نمایش اطلاعات محصوالت فروشگاه است.
این برنامه برای سوپرمارکت‌ها، هایپرمارکت‌ها، فروشگاه‌های لوازم تحریر، آرایشی، لبنیات، مواد غذایی و تقریباً هر نوع فروشگاه دیگری طراحی شده است.

## ✨ ویژگی‌های اصلی

- ✅ ثبت محصول (نام، قیمت، بارکد، کد، دسته‌بندی، توضیحات، تصویر)
- ✅ اسکن بارکد با دوربین
- ✅ جستجوی لحظه‌ای (Realtime) بر اساس نام، بارکد و کد محصول
- ✅ دسته‌بندی محصوالت (۲۲ دسته پیش‌فرض + امکان افزودن)
- ✅ ذخیره تصاویر محصوالت با کیفیت بالا
- ✅ کاملاً آفلاین (بدون نیاز به اینترنت)
- ✅ اجرا روی اندروید و ویندوز
- ✅ ارتباط گوشی و کامپیوتر از طریق شبکه لوکال
- ✅ نسخه پشتیبان و بازیابی
- ✅ شخصی‌سازی کامل ظاهر (رنگ، فونت، تم)
- ✅ حالت تاریک / روشن
- ✅ انیمیشن‌های نرم و حرفه‌ای
- ✅ طراحی Glassmorphism + Material Design 3
- ✅ تاریخچه تغییرات قیمت
- ✅ پشتیبانی از ۱۰۰,۰۰۰+ محصول

## 🏗️ معماری

پروژه از **Clean Architecture** با سه لایه استفاده می‌کند:

```
┌─────────────────────────────────────┐
│        Presentation (UI)            │  ← صفحات، ویجت‌ها، Provider ها
├─────────────────────────────────────┤
│           Domain (منطق)             │  ← Entity ها، UseCase ها، اینترفیس‌ها
├─────────────────────────────────────┤
│            Data (داده)              │  ← مدل‌ها، ریپازیتوری‌ها، DataSource ها
└─────────────────────────────────────┘
```

## 🎨 طراحی

## 📱 صفحات

2. **Home** - جستجو + لیست محصوالت + دکمه افزودن
3. **Add/Edit Product** - فرم ثبت محصول
4. **Product Detail** - جزئیات کامل + ویرایش/حذف
